## FSD TL;DR
```shell
└── src/
    ├── app/                    # App init logic
    ├── processes/              # (Optional) Processes, flowing over the pages
    ├── pages/                  # App pages
    ├── widgets/                # (Optional) Independent content blocks
    ├── features/               # User stories
    ├── entities/               # Business logic
    └── shared/                 # Utilities and common modules
```
